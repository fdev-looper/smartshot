🚀 SmartShot

Your smart image download tracker — right inside Chrome

🧠 What is SmartShot?

SmartShot is a lightweight Chrome extension that monitors every image you download. It logs useful metadata like:

  • 🖼️ Filename of the image
  • 🌐 URL of the image source
  • 🧾 Title & URL of the tab you downloaded it from
  • ⏱️ Timestamp of when it was downloaded

Perfect for researchers, journalists, or curious folks who like to track image sources without hassle.

🔧 Project Files Explained

📄 manifest.json: Defines permissions, background scripts, and extension metadata.
🧠 background.js: Core brain that listens to downloads and stores metadata in Chrome storage.
🖼️ popup.html + popup.js: UI to view your image logs, filter them by filename, time, or URL.
🔒 security.js (optional): Prevents tampering or modifications to your extension logic. Obfuscation/locks can be added here.

🎯 Why I Built SmartShot

Ever downloaded a meme or graphic and forgot where it came from? SmartShot ensures you always know the "when", "where", and "what" behind every image you save. Ideal for:

  • 🧑‍💻 Developers documenting UI changes
  • 🧑‍🏫 Teachers & trainers collecting visual material
  • 👨‍🔬 Researchers saving scientific graphs or images
