Installation

Up and running
in 60 seconds.

SmartShot isn't on the Chrome Web Store — it lives here on GitHub. Install it in four easy steps.

1
Download the code
Clone or download the ZIP from GitHub and extract it to a folder.
git clone https://github.com/fdev-looper/smartshot.git
2
Open Chrome Extensions
Navigate to chrome://extensions in your browser.
3
Enable Developer Mode
Toggle "Developer mode" ON in the top-right corner of the extensions page.
4
Load the extension
Click "Load unpacked" → select the smartshot folder. Done.

Privacy

Your data never leaves your browser.

SmartShot uses chrome.storage.local — your device's own storage. No servers. No accounts. No telemetry. Period.

Zero external requests
SmartShot makes no network calls whatsoever
No account needed
Install and use — no sign-up, no email, nothing
Delete anytime
Clear all history in one click — gone immediately
