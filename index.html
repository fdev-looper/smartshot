🚀 SmartShot

Your lightweight companion for tracking every image download — automatically.

🧠 What is SmartShot?

SmartShot is a simple yet powerful Chrome extension that keeps track of every image you download — so you never lose context again.

Whether you're collecting memes, reference images, research graphics, or just saving cool stuff from the web — SmartShot quietly logs important details behind each download:

  • 🖼️ The filename of the image
  • 🌐 The source URL of the image
  • 🧾 The tab title and tab URL at download time
  • ⏱️ The exact timestamp of when it was downloaded

It's like a digital memory for your downloads — made for curious minds, content hoarders, and organized nerds alike.

🔍 Why SmartShot Exists

Ever downloaded an image and then struggled to remember where you got it from? Yeah, same here. That’s exactly why I built SmartShot.

The extension exists to answer one core question: "Where did I get this image from and when?"

No more messy folders with mystery files. With SmartShot, you get peace of mind, better tracking, and full context every single time.

It's great for:

  • 🧑‍💻 Developers capturing UI screenshots
  • 👩‍🏫 Teachers or trainers saving examples
  • 👨‍🔬 Researchers collecting image-based data
  • 😎 Casual users who want control over their downloads

📁 Breakdown of Project Files

📄 manifest.json: The brain map of the extension — it defines permissions, background behavior, and links everything together.
🧠 background.js: The actual engine. It listens to image downloads and logs the metadata in Chrome’s storage.
🖼️ popup.html + popup.js: The front-end UI. Here’s where you can view, search, and filter your image log — all inside the extension popup.
🔒 security.js (optional): Add-on file to prevent users from tampering with the logic. Optional layer of obfuscation and control.
